HTML

HTML é a linguagem de marcação usada para criar a estrutura de um site ou aplicação web. Você usará HTML para criar a estrutura básica da sua página, incluindo cabeçalhos, parágrafos, listas, imagens, formulários e muito mais. HTML é relativamente fácil de aprender, e é uma ótima linguagem para começar a aprender a criar páginas web.

CSS

CSS é a linguagem de estilo usada para adicionar estilos visuais ao seu site ou aplicação web. Você usará CSS para definir a aparência visual de seus elementos HTML, como cores, fontes, tamanhos, espaçamento, posicionamento e muito mais. CSS pode ser um pouco mais desafiador do que HTML, mas é uma habilidade essencial para qualquer desenvolvedor web.

JavaScript

JavaScript é uma linguagem de programação que permite criar interatividade e dinamismo em seu site ou aplicação web. Você usará JavaScript para adicionar recursos interativos, como botões, animações, formulários dinâmicos, efeitos visuais e muito mais. JavaScript é uma linguagem poderosa e versátil, mas também pode ser um pouco mais desafiadora do que HTML e CSS.
